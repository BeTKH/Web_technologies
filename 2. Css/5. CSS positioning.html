CSS POSITIONING

CSS Position The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute or sticky).

Types of CSS positioning styles:

  1. Relative
  2. Absolute
  3. Fixed - Typically used to create a menu that stays on the page while the user scrolls through the content of the page.
  4. z-index: to bring forward/backward overlapping objects
